Intro to Mongoose

## Setup

### install 

=> npm init -y

=> npm install express --save

=> npm i nodemon --save-dev

=> npm i mongoose

<!-- server -->
=> npm i dotenv

### Modify scripts

package.json
"scripts":
    "start": "node src/index.js",
    "dev": "nodemon src/index.js",
    "seed": "node src"
